SharePoint lists can hold millions of items technically, but performance degrades past the 5,000-item view threshold. Warnings appear at 4,000 items to give a safe buffer to archive before hitting it.

cloud_done SharePoint Connection

Data syncs from SharePoint every 60 seconds. Use the buttons below for manual operations.

backup Backup & Restore

Checking...

Auto-backup runs three times daily (Night: 12-7 AM, Day: 7 AM-3 PM, Evening: 3-12 PM) when anyone opens the Shift Hub. Backups are stored as JSON in the SharePoint document library.

Restore from Backup

Upload a JSON backup to restore data. Existing items with matching IDs will be updated. New items will be created. No data is deleted.
